# PROMPT_REGISTRY (Single Source of Truth)

## 원칙
- 새창 프롬프트는 반드시 이 파일에서만 복사한다.
- 버전 = 의미/명령/출력 구조 변경, 프로필 = 길이/강도/출력 개수.
- 태그는 항상 포함: [vX.X-PROFILE 제공] ... [vX.X-PROFILE 제공완료]

## 고정 헤더(항상 포함 6줄)
1) 파일 수정/생성/포맷팅 금지
2) 출력은 지정 섹션만
3) 중간 생략/요약/의역 금지
4) 프로필 기준 명시(예: LongSet)
5) 시작/끝 마커(--- 시작 --- / --- 끝 ---)
6) 상단/하단 태그 포함
